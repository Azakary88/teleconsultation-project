Frontend React complet pour l'application de téléconsultation.

Installation:
1. Node 18+
2. npm install
3. create a .env file with VITE_API_BASE=http://localhost:3000
4. npm run dev

Notes:
- Le frontend suppose les endpoints découverts dans ton backend.
- Si GET /api/appointments manque, ajoute-le côté backend pour lister les RDV d'un utilisateur.
- Pour la vidéo mobile/web, j'ai prévu une WebView Jitsi dans le scaffold mobile Expo (si besoin).
