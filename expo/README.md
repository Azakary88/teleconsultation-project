Expo Teleconsult App - Scaffold complet (écrans inclus)

Installation:
1. Node 18+
2. npm install
3. Modifier app.config.js extra.apiBase si nécessaire (ex: http://192.168.x.y:3000)
4. npx expo start

Tester:
- Utilise Expo Go pour ouvrir l'app sur un téléphone réel ou exécute un émulateur Android.
- Les écrans fournis couvrent l'auth, listing médecins, création RDV, listing RDV, et vidéo via Jitsi WebView.

Remarques:
- Assure-toi que ton backend est accessible depuis l'app (emulator vs device addresses).
- Si tu veux que j'ajoute le refresh token auto ou notifications push, demande-le.
